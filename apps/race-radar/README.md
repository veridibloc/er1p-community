# Race Radar

Public-facing web application for tracking live races and exploring race history. Built for running enthusiasts to view active races in real-time, monitor leaderboards, and explore historical race data.

## Stack

Next.js 15, React 19, Tailwind CSS, running on Bun.

## Development

```bash
# Start development server (runs on port 3003)
bun dev
```

Development server runs at **http://localhost:3003**

## Build

```bash
# Build for production
bun run build

# Start production server
bun start
```
